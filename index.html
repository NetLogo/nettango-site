Block-based Programming
for NetLogo Web

NetTango is a blocks-based programming environment for NetLogo designed to make it easier than ever to engage in agent-based modeling with little or no programming experience.
Launch NetTango Builder Read the Tutorial

The NetTango Ecosystem

Define the blocks, or play with the model. Two interfaces for different needs.
NetTango Desktop
Older desktop version of NetTango that works with NetLogo Desktop models.
NetTango Builder
NetTango builder is an interface for defining blocks and linking them to existing NetLogo Web models.
NetTango Player
NetTango player provides the model for end-users to experiment with the models behavior using domain-specific blocks.

Getting Started

Collaboration & Open Source

NetTango Web is developed as a collaboration between the Center for Connected Learning and the TIDAL lab, both at Northwestern University.

NetTango Web is open source software. The NetTango Web builder is part of the Galapagos project for NetLogo Web. The NetTango blocks interface has its own repository.

Domain-centric block-based programming, powered by NetLogo Web for agent-based modeling and simulations used by researchers and educators.
